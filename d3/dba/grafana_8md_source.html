17 
18 + Open port 3000 **(as root)**
19 ~~~
20 firewall-cmd --zone=public --add-port 3000/tcp --permanent
21 firewall-cmd --reload
22 ~~~
23 
24 + Change default `admin_user` and `admin_password`: `/etc/grafana/grafana.ini`. **(as root)**
25 
26 See more regarding configuration file in the official documentation: http://docs.grafana.org/installation/configuration/
27 
28 + (Enable SSL)
29  + Set protocol to `https`, `ssl_mode` to `skip-verify` in configuration file
30  + Generate private key and certificate via [CERN Certification Authority](https://ca.cern.ch/ca/host/HostCertificates.aspx)
31  + Set `cert_file` and `cert_key` value in configuration file
32 
33 + (Configure LDAP-based login: `/etc/grafana/ldap.toml`)
34 See official documentation at the Grafana webpage: http://docs.grafana.org/installation/ldap/
